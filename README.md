### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/EmilMorua/python-project-50.git
   ```
2. Installing dependencies
   ```sh
   make install
   ```
3. Building a package
   ```sh
   make build
   ```
4. Usage
   ```sh
   gendiff [options] file_path1 file_path2
   ```

Available Options:
   ```sh
   --format, -f - output format: stylish (default), plain, json
   ```
